## üöÄ How to Start

### How to Configure

You can open the settings menu a few different ways:

- Chat Command: **`/nt config`** or **`/ntconfig`**
- **Minimap Button**
- **WoW Addon Compartment** (Left-click to configure)

### Quick Commands

- **`/nt version`** - Display current addon version
- **`/nt`** - Show help with available commands
- **`/rl`** or **`/reloadui`** - Fast UI reload

## üì¶ How to Install

Simply install the addon through your preferred addon manager (CurseForge, Wago, etc.) or manually extract it to your `Interface/AddOns` folder.
