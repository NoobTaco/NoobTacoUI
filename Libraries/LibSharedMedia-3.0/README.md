# LibSharedMedia-3.0 (NoobTaco Mirror)

This is a GitHub mirror of the official `LibSharedMedia-3.0` library, maintained for the **NoobTacoUI** ecosystem.

## Purpose
- **High Availability**: Provides a reliable GitHub-based source for the library.
- **Workflow Integration**: Compatible with the BigWigsMods packager using the `externals` section in `.pkgmeta`.
- **Preservation**: Tracks the official Subversion repository history while allowing for modern Git-based development.

## Official Source
- **WowAce**: [LibSharedMedia-3.0](https://www.wowace.com/projects/libsharedmedia-3-0)
- **SVN**: `https://repos.wowace.com/wow/libsharedmedia-3-0/trunk`
